Setelah login Anda menemui halaman seperti ini:


Data Kereta

1. Tambah kereta

Tekan tombol tambah data kereta dan mengisi Nama kereta, deskripsi kereta, dan tipe kereta lalu tekan SAVE

2. Edit Kereta

Pilih opsi edit untuk mengedit/merubah data kereta yang telah dibuat

3. Hapus Kereta.

Pilih opsi sampah untuk menghapus data kereta yang telah dibuat


Data Gerbong

Klik pada nama kereta untuk masuk kehalam Data gerbong

1. Tambah Gerbong

Tekan tombol TAMBAH DATA GERBONG dan mengisi Nama gerbong dan jumlah kursi lalu tekan SAVE

2. Edit Gerbong

Pilih opsi EDIT untuk mengedit/merubah data gerbong yang telah dibuat

3. Hapus Gerbong

Pilih opsi SAMPAH untuk menghapus datagerbong yang telah dibuat


Data Kursi

1. Tambah kursi

Tekan ikon PLUS pada halaman yang berada dalam gerbong untuk menambah kursi dengan memasukkan nomor kursi

2. Edit Kursi

Pilih opsi EDIT untuk mengedit/merubah data kursi yang telah dibuat

3. Hapus Kursi

Pilih opsi SAMPAH untuk menghapus data kursi yang telah dibuat


Data Admin

1. Tambah Admin

Tekan tombol TAMBAH DATA ADMIN untuk menambahkan admin dengan memasukkan Biodata dan tekan SAVE untuk menyimpan

2. Edit Admin

Pilih opsi EDIT untuk mengedit/merubah data admin yang telah dibuat

3. Hapus Admin

Pilih opsi SAMPAH untuk menghapus data admin yang telah dibuat

4. Reset Password

Pilih opsi RESET PASSWORD untuk mengganti passoword admin apabila mau


Data Pelanggan

1. Tambah Pelanggan

Tekan tombol TAMBAH DATA PELANGGAN untuk menambahkan pelanggan dengan memasukkan Biodata dan tekan SAVE untuk menyimpan

2. Edit Pelanggan

Pilih opsi EDIT untuk mengedit/merubah data pelanggan yang telah dibuat

3. Hapus Pelanggan

Pilih opsi SAMPAH untuk menghapus data pelanggan yang telah dibuat

4. Reset Password

Pilih opsi RESET PASSWORD untuk mengganti passoword Pelanggan apabila mau


Data Jadwal

1. Tambah Jadwal

Tekan tombol TAMBAH DATA JADWAL untuk menambahkan data keberangkatan, harga, jenis kereta, dan waktu dan tekan SAVE untuk menyimpan

2. Edit Jadwal

Pilih opsi EDIT untuk mengedit/merubah data jadwal yang telah dibuat

3. Hapus Jadwal

Pilih opsi SAMPAH untuk menghapus data jadwal yang telah dibuat
